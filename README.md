# UdemyUnity2D-TwinSticks2.5D
The chapter from 'Complete C# Unity Developer 2D' Udemy course discussing Twin Sticks 2.5D

Focus was not on making a game, it was mainly focused on source control and handling multiple input sources ie mobile, controller, keyboard etc.
Replays and pausing were also introduced. They also demonstrated how to swing the camera around tying it to the right stick.
